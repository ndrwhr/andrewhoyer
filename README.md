###Building:

Ensure that all required gems (using bundler) and node_modules (using yarn) are installed:

`bundle install`
`yarn install`

###Developing:

* `bundle exec jekyll server --watch`
* `yarn run watch`

###Deploying:

Run `./deploy.sh`.

###Dependancies:
